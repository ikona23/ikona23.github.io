What mobile first design is, and why it's important.

Mobile has a very limited space for displaying content. Because mobile screen has limited space, it is very important to start design and code website from the mobile view. We will use less content.

Wireframe is scatch of your website before you even start to code. Simle visual representation of elements on a website. Wireframe is very important for development. It lets you create hierarchy. Simplify communication between developer and client.
