Explain Map(), filter & reduce()

AMap()is useful if you want to apply a transformation to every item in the array and collect up the results in a new array. Sometimes the map function is not enough. Sometimes we want to apply changes only to some items. Items that match only certain tasks. For this purpose we use a filter and reduce() is used when you want to find a cumulative or concatenated value based on elements across the array. (This method reduces the array to a single value).
